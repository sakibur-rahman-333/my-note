# My Note Project

## Running process

1. Create a new folder.
   
2. Open terminal on that location and type --> `git clone git@github.com:sakibur-rahman-333/my-note.git`
   
3. After successfully cloning repo, install all the dependencies. Type on the terminal --> `npm i`
   
4. After installing dependencies type --> `npm start`
